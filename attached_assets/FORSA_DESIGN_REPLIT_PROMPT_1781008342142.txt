## TECH STACK
- Frontend: React + React Router (dla /en/, /pl/) + Tailwind CSS
  ALTERNATYWA: HTML5 / CSS3 / Vanilla JavaScript z language context object
- Language State: localStorage untuk persistence
- Responsive: Mobile-first design, tested on mobile/tablet/desktop
- Animations: Subtle fade-in on scroll, gold hover effects on buttons/links, soft transitions between languages
- Contact Form: Frontend validation (name, email, message required) + backend submission
- Icons: Rocket icon z loga (animated subtle float na hero), service icons

## ANIMACJE & INTERAKCJE
- Fade-in on scroll dla sekcji
- Gold accent (#D4A574) na hover dla buttons, links
- Rocket icon: gentle floating animation na hero section
- Language switcher: smooth fade transition przy zmianie języka
- Smooth scroll behavior
- CTA buttons: hover state z shadow/scale

## SEO & META
- Meta tags dla każdego języka (title, description, og:locale)
- Semantic HTML (header, nav, main, section, footer)
- hreflang tags w <head> (link rel="alternate" hreflang="en" / "pl")
- og:image, og:title, og:description
- Alt text na wszystkie images
- Fast load time (optimize images, minify CSS/JS)
